# gas_repeat_skipper

Google Apps Scripts 土日祝日をスキップする繰り返し

# 概要

- Google カレンダーに繰り返しスケジュールを作成します。  
  曜日や祝日に応じキャンセル又は繰り上げたり先延ばしにすることができます。

# 使い方

1. Apps Script プロジェクトを作成します。
1. `caller` `main` という名前のスクリプト ファイルを作成します。
1. 作成された `caller.gs` 及び `main.gs` に本リポジトリの同じ名前のファイルの内容を貼り付けます。
   - [caller.gs](./caller.gs)
   - [main.gs](./main.gs)
1. ファイルから **caller.gs** を選択し、`schedule1` ～ `schedule4` のいずれかを実行するとスケジュールが作成されます。
   - 必要に応じ main 関数に渡す引数や、あらかじめ用意された変数の代入値を変更して下さい。
   - 関数の引数は JSDoc コメントにおおまかな機能が記述されています。

# 注意

- 十分なテストは行っておりませんので、支障のないスケジュールを設定してお試し下さい。  
  本スクリプトにより生じた損害について、作者は一切の責任を負いません。

# 補足

- 通知は既定で設定されます。不要又は変更したい場合は、どれか1つのスケジュールから通知を削除もしくは変更し、  
  **保存** をクリックした後に表示される **定期的な予定の編集** から **すべての予定** を選択して下さい。
